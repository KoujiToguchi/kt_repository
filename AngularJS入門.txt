環境構築のための設定

1	node.jsを本家サイト(http://nodejs.org/download/)からDL,インストール		
	（Next連打でＯＫ）				
							
							
	proxy の設定（コマンドプロンプトで実行、管理者権限で起動すること)
							
	npm config set proxy http://proxy.occ.co.jp:8080
	npm config set https-proxy http://proxy.occ.co.jp:8080
	npm config set registry http://registry.npmjs.org/
	set HTTPS_PROXY=http://proxy.occ.co.jp:8080		
								
2	typescriptのインストール(npm install typescript -g)
							
3	Bowerのインストール(npm install bower -g)
						
4	gruntのインストール(npm install grunt -g)
						
5	gitExtensionsのインストール		
	http://codezine.jp/article/detail/7077?p=2
						
6	gitCloneの作成（git clone https://github.com/Polaris-Framework/polaris-web.git）
	(gitの許可を得るためにGitHubユーザー情報入力が必要だが島尻のアカウントでないとダメ（許可を得てない))
	(gitCloneの作成箇所はC直下だとダメ、ユーザディレクトリの直下が良い？)

7	ディレクトリをpolaris-webに移動

8	pacage.jsonに書かれてる内容をnpmでインストールする(npm install)
											
9	bower.jsonの記述内容をbowerでインストールする。(bower install)			
								
10	gruntを実行しサーバを起動する。(grunt serve)

11  	http://localhost:9001に接続。画面が表示されること。
